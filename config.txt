* Para rodar o projeto
- npm run dev

* Para criar o projeto na pasta 'React apoio' abri com code e no terminal gitbash rodei
- npm create vite@latest ecommerce-react

- npm i 

- npm i react-icons react-router-dom
- link de icons https://react-icons.github.io/react-icons/icons?name=bs

* Para instalar o bootstrap
- npm install bootstrap
- npm install react-bootstrap
- e no main.jsx importar. ex: import 'bootstrap/dist/css/bootstrap.min.css'
- npm i mdb-react-ui-kit (não é necessario no bootstrap, apenas para a timeline é necessario)


* Para instalar o chakara ui
- npm i @chakra-ui/react @emotion/react @emotion/styled framer-motion
